# Controle de Estufa de Hortaliças com Arduino



Este projeto utiliza a plataforma Arduino para criar um sistema de controle de estufa de hortaliças simulado no TinkerCad. O objetivo é monitorar a temperatura ambiente e controlar dispositivos como um motor de ventilador, um LED e uma buzina para simular um ambiente ideal para o cultivo de plantas.



## Componentes Utilizados:

- Arduino Uno

- Sensor de Temperatura (termistor)

- Motor de Ventilador

- LED

- Buzina



## Funcionalidades Implementadas:



1. **Leitura da Temperatura:**

  - Utiliza um sensor de temperatura para monitorar a temperatura ambiente.



2. **Controle do Motor de Ventilador:**

  - Aciona um motor de ventilador quando a temperatura atinge ou ultrapassa 30 °C, ajudando a manter o ambiente da estufa mais fresco.



3. **Alerta de Emergência:**

  - Ativa um LED vermelho e uma buzina caso a temperatura ultrapasse os 50 °C, indicando uma situação de emergência.



## Instruções de Uso:



1. **Configuração do Hardware:**

  - Conecte o sensor de temperatura, motor de ventilador, LED e buzina à placa de ensaio (protoboard) de acordo com o esquema fornecido.



2. **Carregando o Código no Arduino:**

  - Utilize a IDE do Arduino para carregar o código fornecido neste repositório na placa Arduino Uno.



3. **Execução do Projeto:**

  - Após o upload do código, o Arduino começará a monitorar a temperatura ambiente e controlar os dispositivos conforme as condições especificadas.



4. **Ajustes e Personalização:**

  - Modifique o código conforme necessário para se adequar às suas configurações de hardware específicas ou para adicionar funcionalidades adicionais.



## Notas Adicionais:

- Este projeto foi desenvolvido no simulador TinkerCad para fornecer uma simulação virtual de um sistema de controle de estufa. Certifique-se de ajustar as conexões e componentes se estiver utilizando hardware físico.



- Assegure-se de respeitar as especificações elétricas dos componentes utilizados para evitar danos.
